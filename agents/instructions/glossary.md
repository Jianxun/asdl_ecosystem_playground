# Glossary

- **ASDL**: the circuit description language used for design and testbench authoring.
- **Experiment library**: a focused ASDL package under `libs/<experiment_name>/`.
- **Backend**: emitter target syntax/profile (for example `sim.xyce`).
- **PDK assets**: model and primitive definitions under `pdks/`.
- **Normalization**: converting raw simulator outputs into consistent analysis formats.
- **OP inspection**: operating-point data review, typically pivoted by device/parameter.
- **Hydration pack**: instruction files used to quickly bootstrap new agent sessions.
- **Ergonomics issue**: anything that makes authoring/debugging slower, error-prone, or unclear.
- **Reusable snippet**: short, stable ASDL or command pattern worth carrying forward.
- **Legacy simorc**: older orchestration implementation in `asdl_ecosystem/simorc`; reference only, not the target architecture for this playground.
