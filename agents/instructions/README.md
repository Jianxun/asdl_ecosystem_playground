# Agent Instructions

These files are the primary hydration pack for agents working in ASDL Playground.

Start here, in order:

1. `quickstart.md`
2. `workflow.md`
3. `tooling.md`
4. `debugging.md`
5. `snippets.md`
6. `experiment_template.md`
7. `glossary.md`

## Purpose

- Keep agent behavior consistent across sessions.
- Reduce startup friction for ASDL author -> compile -> simulate -> analyze loops.
- Capture ergonomics findings that will inform the next-generation orchestration tool
  (separate from the legacy `asdl_ecosystem/simorc` implementation).
