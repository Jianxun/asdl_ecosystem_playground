# OBJECTIVE

Continuously improve the analog circuit design capability of the ASDL ecosystem by developing and optimizing the agent harness.

## Direction

- Treat harness quality as the primary product.
- Use labs and circuit tasks as evaluation vehicles.
- Convert recurring execution friction into reusable guidance, libraries, and scripts.
- Keep workflows reproducible, explicit, and runnable from repo root.

## Operating Principle

This is a continuous-improvement objective, not a fixed end state. There is no final "done" condition; progress is measured by sustained reductions in friction and expanded ability to execute more challenging analog design tasks.
