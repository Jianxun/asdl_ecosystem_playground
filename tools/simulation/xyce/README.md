# Xyce Tools

Canonical simulation-normalization utilities for playground workflows.

- `raw_to_h5.py`: primary RAW to HDF5 normalization flow.
- `format_xyce_op_csv.py`: pivots Xyce operating-point tables for quick OP inspection.

Legacy wrappers remain in `libs/tb/tb_ota_5t/` for compatibility and print deprecation notices.
