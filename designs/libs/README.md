# Libraries

This directory is the primary authoring location for experiment libraries.

## Convention

- Create one subdirectory per experiment library.
- Keep `.asdl` source in the library directory.
- Write generated outputs to `runs/` instead of committing emitted netlists/waveform artifacts here.

## Existing libraries

- `mosbius_devices/`
- `sw_matrix/`
- `tb/`
- `scratch/`
